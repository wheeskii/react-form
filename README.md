# REACT FORM
A simple TypeScript full stack React + Express project demonstrating basic form handling, validation and client-server communication.


## Overview
This project contains a beginner's work meant for learning purposes practing full-stack TypeScript/Node.js development.
## Technologies Used
|  Frontend  |  Backend  |
|:---:|:---:|
|  React     |  Node.js  |
|  Vite      |  Express  |
|  Axios     |  MySQL    |
|  Zod       |  Zod      |
| Vanilla CSS|  TypeORM  |


## Purpose of this Project
This repo is intended for practicing:
- Full-stack TypeScript workflow
- React form handling
- Sending data to an Express backend
- Basic validation
